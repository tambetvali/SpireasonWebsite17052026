Flatlist — Index

Plain Markdown listings of Laegna, Wave, and equality formats.

Files in this folder

This folder contains simplified, flat Markdown listings of the Laegna number system. Each file presents the numbers in a different format: plain lists, paired Laegna/Wave forms, or equality mappings to decimals. All files are generated from the same underlying number database. 

toc.md             — Short introduction and table of contents for this folder
laegna.md          — Plain Laegna number list, grouped by digit length (R)
multiline.md       — Laegna + Wave + Decimal, multiline format per number
laegnawaves.md     — Wave-only listing, grouped by digit length
laeequality.md     — Laegna-to-decimal equality list (with NULL cases)
waveequality.md    — Wave-to-decimal equality list, grouped by R

These Markdown files provide the simplest, most lightweight access to the number system. They are suitable for quick browsing, scripting, or external tools that prefer plain text formats.

A future update will add HTML versions of each file, as well as a viewer for complete.json in the parent folder.
