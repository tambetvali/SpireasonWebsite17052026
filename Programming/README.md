# Programming — Spireason Editions  
This folder contains the **Spireason‑formatted versions** of two foundational Laegna texts:

- **Book 1 — Mission**  
- **Book 2 — Programming Matters**

These versions are adapted for the Spireason website:  
clean layout, consistent color logic, and easy reading on mobile and desktop.  
They are intended for **local browsing**, while the **canonical originals** remain preserved in the LaeArve repository.

---

## 📘 Book 1 — Mission  
**Local version:**  
`Book1Mission.html` (this folder)

**Original canonical version:**  
https://github.com/tambetvali/LaeArve/tree/main/History/Laegna/Books/Book1Mission

**About this book:**  
*Mission* is the introductory text of Laegna.  
It explains:
- the purpose of Laegna  
- the philosophical and structural motivations  
- the early symbolic framework for programming  
- how the system relates to human reasoning and intentionality

The original repository includes multiple formats (Markdown, PDF, and source files),  
while the Spireason version is optimized for web presentation.

## 📗 Book 2 — Programming Matters  
**Local version:**  
`Book2ProgrammingMatters.html` (this folder)

**Original canonical version:**  
https://github.com/tambetvali/LaeArve/tree/main/History/Laegna/Books/Book2ProgrammingMatters

**About this book:**  
*Programming Matters* introduces the computational and structural logic of Laegna.  
It covers:
- symbolic programming principles  
- the role of structure, order, and intentionality  
- how Laegna logic translates into practical computation  
- early examples of symbolic encoding

The original repository contains the full set of formats (Markdown, PDF, and source),  
while the Spireason version is adapted for interactive reading.
