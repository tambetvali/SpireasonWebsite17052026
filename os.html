🔧 How to Make These Systems Your Own

Every Laegna tool can become your tool: same structure, new story.

A simple path to reuse:

  • 1. Choose a repository that matches what you want: a calculator (Exponential Whispers, Theorem Theater, Exponometer), a visualization (Mind Spark Visions, Chem Visions AI), or an engine (Nexus Warp, Mind Sculptor Engine).
  • 2. Fork it on GitHub (or download the .zip using the buttons above) so you have your own copy. This is your personal Laegna branch.
  • 3. Open it in an AI web‑app builder — preferably lovable.dev, or alternatively bolt.new or similar. Ask the system explicitly:
    “This is a Laegna‑style calculator / visualization. Help me change the labels, the skin, and the explanations so it matches my topic: [your topic].”
  • 4. Remove what you do not need. If a page has three calculators and you only need one, delete the others. If a visualization has too many panels, simplify it. The structure is there to be edited, not worshipped.
  • 5. Add new calculators or tools by copying an existing one: duplicate a component, rename the functions, change the formulas and the text. For example, turn an octave‑based Exponometer into a “habit growth” calculator simply by changing the variables and the narrative.
  • 6. Re‑skin the experience. Change colors, fonts, icons and wording so that the tool feels like it belongs to your project, your community, your spiritual or scientific language. Laegna is a grammar; you are free to write new sentences.
  • 7. Re‑deploy. Once modified, re‑deploy via lovable.dev or your chosen builder. You now have a new calculator, visualization or engine — grown from Laegna soil, but speaking in your own voice.

If you get lost, open the original SpiReason main page and follow the links back: each live page has a corresponding repository here. The loop is intentional: experience → source → remix → new experience.
