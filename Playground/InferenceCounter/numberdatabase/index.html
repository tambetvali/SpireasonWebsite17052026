Number Database — Index

Generated extended dataset for the Inference Counter.

Files in this folder

This folder contains the generated number database used by the Inference Counter. The primary file is complete.json, which merges numbers.json and numbers.bin.json and applies all extensions defined in extensions.py. 

complete.json     — Full extended number database (main file)
flatlist/         — Folder containing flat Markdown number lists

The complete.json file is the authoritative extended dataset. The flatlist folder will contain simplified Markdown listings for easier browsing and lightweight use.
